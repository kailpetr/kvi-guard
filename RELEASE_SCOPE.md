# KVI Guard Scope

KVI Guard intentionally avoids:

- universal truth verification
- AGI architectures
- autonomous agents
- massive knowledge graphs
- persistent orchestration systems
- speculative cognition frameworks
- heavyweight dependency stacks

The project focuses on:

- lightweight runtime verification
- inference stability analysis
- semantic consistency detection
- hallucination pressure signals
- interpretable telemetry
- deployable local middleware

The architecture is intentionally constrained to remain:

- understandable
- benchmarkable
- lightweight
- locally runnable
- maintainable
